GDP Gains NPZ Format

Raw selected gain/bandpass samples saved by gdp-stats --gains.

Documentation Tree

  • Main documentation
  • gdp-setup: initialize and remember runtime and source-data locations.
  • gdp-util: read metadata from CASA gain, bandpass, or MeasurementSet tables.
  • gdp-stats: compute antenna-wise gain and bandpass products.
  • gdp-plot: plot GDP NPZ products and create missing products with gdp-stats.
  • gdp-plan-run: run GDP command intents from a plan file.

Product Formats

  • gains NPZ
  • stats NPZ
  • KS NPZ
  • self-corr NPZ
  • cross-corr NPZ

Standard Name

gdp-gains-<mode>-<scan>.npz

Example: gdp-gains-gain-scan1.npz. With --combine-scans: gdp-gains-gain-allscans.npz.

Arrays

ArrayDescription
header_jsonJSON metadata with format, version, input table, mode, data column, flag policy, channel selection, and column names.
scanScan number for each saved sample.
antennaAntenna ID for each saved sample.
stokesStokes/polarization index.
channelChannel index after selection.
rowOriginal CASA table row index.
timeCASA TIME value for the original table row, in seconds.
intervalCASA INTERVAL value for the original table row, in seconds. Used to scale gain colormap time bins.
realReal component value; flagged samples become NaN when flags are used.
imagImaginary component value; flagged samples become NaN when flags are used.
flaggedBoolean flag state for each sample.
