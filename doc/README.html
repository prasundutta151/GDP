GDP Documentation

GDP (Gain Diagnostic Product) is a command-line toolkit for gain and bandpass diagnostics, data products, plotting, diagnosis, and flag products.

Documentation Tree

  • Main documentation
  • gdp-setup: initialize and remember runtime and source-data locations.
  • gdp-util: read metadata from CASA gain, bandpass, or MeasurementSet tables.
  • gdp-stats: compute antenna-wise gain and bandpass products.
  • gdp-plot: plot GDP NPZ products and create missing products with gdp-stats.
  • gdp-plan-run: run GDP command intents from a plan file.

Product Formats

  • gains NPZ
  • stats NPZ
  • KS NPZ
  • self-corr NPZ
  • cross-corr NPZ

Quick Start

script/gdp-setup --gain-table /path/to/gain/table.g --bandpass-table /path/to/bandpass/table.b --rundir /path/to/rundir  [save separate gain and bandpass defaults plus runtime directory]
script/gdp-setup --show  [print the saved GDP setup]
script/gdp-util /path/to/table.g --header  [inspect table metadata]
script/gdp-stats --stats --csv  [compute stats and optional CSV from the saved input]
script/gdp-stats --stats --combine-scans  [write one combined allscans stats product]
script/gdp-stats --mode gain --gain-colormap  [create gains NPZ and Real-1/Imag colormap plots with elapsed seconds on the y axis]
script/gdp-plot --mode gain -pmode gain-hist  [plot AntStat-style gain histograms from gains NPZ]
script/gdp-plot --mode gain -pmode antenna 0,1,2  [plot selected antennas with Stokes side by side]
script/gdp-plot --mode gain -pmode stats  [plot stats, creating the stats NPZ first if needed]
script/gdp-plan-run pipelines/sample_stats.plan --dry-run  [preview a pipeline plan without running it]
