GDP Stats NPZ Format

Antenna-wise gain/bandpass statistics saved by gdp-stats --stats.

Documentation Tree

  • Main documentation
  • gdp-setup: initialize and remember runtime and source-data locations.
  • gdp-util: read metadata from CASA gain, bandpass, or MeasurementSet tables.
  • gdp-stats: compute antenna-wise gain and bandpass products.
  • gdp-plot: plot GDP NPZ products and create missing products with gdp-stats.
  • gdp-plan-run: run GDP command intents from a plan file.

Product Formats

  • gains NPZ
  • stats NPZ
  • KS NPZ
  • self-corr NPZ
  • cross-corr NPZ

Standard Name

gdp-stats-<mode>-<scan>.npz

Example: gdp-stats-gain-scan1.npz. With --combine-scans: gdp-stats-gain-allscans.npz.

Arrays

ArrayDescription
header_jsonJSON metadata with format, version, input table, mode, data column, flag policy, mean-subtraction policy, channel selection, and axis descriptions.
scan_numbersScan axis values.
antennasAntenna axis values.
stokesStokes/polarization indices.
componentsComponent axis: real, imag.
metricsMetric axis: mean, std, skew, kurtosis, median, mad, min, max.
statsStatistics array with shape (scan, antenna, stokes, component, metric).
valid_countFinite sample count with shape (scan, antenna, stokes, component).
total_countTotal selected sample count with shape (scan, antenna, stokes, component).
flagged_countFlagged selected sample count with shape (scan, antenna, stokes, component).

The statistics follow the AntStat plot convention: real samples are real(gain) - 1, imaginary samples are imag(gain), mean/std/median/MAD/min/max are saved in percent, and skew/kurtosis are dimensionless.

Read This Product In Python

from pathlib import Path
import json
import numpy as np

path = Path("rundir/data-product/stats/npz/gdp-stats-gain-scan2.npz")
with np.load(path, allow_pickle=True) as z:
    header = json.loads(str(z["header_json"]))
    scans = z["scan_numbers"].astype(int)
    antennas = z["antennas"].astype(int)
    stokes = z["stokes"].astype(int)
    components = z["components"].astype(str)
    metrics = z["metrics"].astype(str)
    stats = z["stats"].astype(float)
    valid_count = z["valid_count"].astype(int)

mean_i = list(metrics).index("mean")
real_i = list(components).index("real")
mean_real = stats[:, :, :, real_i, mean_i]
print("shape:", stats.shape)
print("mean real percent by scan/antenna/stokes:", mean_real.shape)

Reading this product gives a dense array with axes (scan, antenna, stokes, component, metric). The axis-label arrays tell you which scan, antenna, Stokes, component, and metric each index represents. Use valid_count, total_count, and flagged_count to check how much data contributed to each statistic.
