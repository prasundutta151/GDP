GDP Self-Correlation NPZ Format

Normalized structure-function curves saved by gdp-stats --self-corr.

Documentation Tree

  • Main documentation
  • gdp-setup: initialize and remember runtime and source-data locations.
  • gdp-util: read metadata from CASA gain, bandpass, or MeasurementSet tables.
  • gdp-stats: compute antenna-wise gain and bandpass products.
  • gdp-plot: plot GDP NPZ products and create missing products with gdp-stats.
  • gdp-plan-run: run GDP command intents from a plan file.

Product Formats

  • gains NPZ
  • stats NPZ
  • KS NPZ
  • self-corr NPZ
  • cross-corr NPZ

Standard Name

gdp-self-corr-<mode>-<scan>.npz

Example: gdp-self-corr-gain-scan1.npz. With --combine-scans: gdp-self-corr-gain-allscans.npz.

Arrays

ArrayDescription
header_jsonJSON metadata with correlation settings such as lag binning, bin count, and jackknife folds.
scanScan number for each lag-bin row.
antennaAntenna ID for each lag-bin row.
stokesStokes/polarization index.
componentComponent label: real or imag.
tauLag-bin center.
s2Normalized order-2 structure-function value.
errJackknife uncertainty when requested.
countPair count contributing to the lag bin.

If all samples for an antenna/stokes/component are flagged or non-finite, GDP keeps the expected lag-bin rows with s2 and err set to NaN and count set to zero. Plotting code renders those bins as blank or white cells.

Read This Product In Python

from pathlib import Path
import json
import numpy as np

path = Path("rundir/data-product/self-corr/npz/gdp-self-corr-gain-scan2.npz")
with np.load(path, allow_pickle=True) as z:
    header = json.loads(str(z["header_json"]))
    scan = z["scan"].astype(int)
    antenna = z["antenna"].astype(int)
    stokes = z["stokes"].astype(int)
    component = z["component"].astype(str)
    tau = z["tau"].astype(float)
    s2 = z["s2"].astype(float)
    err = z["err"].astype(float)
    count = z["count"].astype(int)

mask = (antenna == 0) & (stokes == 0) & (component == "real")
print("antenna 0 real tau:", tau[mask])
print("antenna 0 real S2:", s2[mask])

Reading this product gives a long table of structure-function lag bins. Each row belongs to one scan, antenna, Stokes, component, and lag tau. Use s2 for the normalized structure-function value, err for jackknife uncertainty, and count to see how many pairs contributed to the bin.
