GDP Cross-Correlation NPZ Format

Cross structure-function correlation-time products saved by gdp-stats --cross-corr.

Documentation Tree

  • Main documentation
  • gdp-setup: initialize and remember runtime and source-data locations.
  • gdp-util: read metadata from CASA gain, bandpass, or MeasurementSet tables.
  • gdp-stats: compute antenna-wise gain and bandpass products.
  • gdp-plot: plot GDP NPZ products and create missing products with gdp-stats.
  • gdp-plan-run: run GDP command intents from a plan file.

Product Formats

  • gains NPZ
  • stats NPZ
  • KS NPZ
  • self-corr NPZ
  • cross-corr NPZ

Standard Name

gdp-cross-corr-<mode>-<scan>.npz

Example: gdp-cross-corr-gain-scan1.npz. With --combine-scans: gdp-cross-corr-gain-allscans.npz.

Output Location

By default this product is written under rundir/data-product/cross-corr/npz/, or under the configured runtime directory from gdp-setup.

Arrays

ArrayDescription
header_jsonJSON metadata with product name, mode, scan selection, column order, and correlation settings.
scanScan number for the antenna-pair measurement.
ant1First antenna ID in the baseline-like antenna pair.
ant2Second antenna ID in the antenna pair.
stokesStokes/polarization index used for the comparison.
componentsGain components used in the comparison, such as real-real, real-imag, imag-real, or imag-imag.
thresholdStructure-function threshold used to extract the correlation time. Default: 0.7.
tcorrEstimated cross-correlation time from the first valid lag bin crossing the threshold. Written as nan when no valid crossing is found.
nbin_validNumber of lag bins with finite structure-function values.
npairTotal sample-pair count contributing to the antenna-pair estimate.

Optional CSV

Use --csv or --output-cross-corr-csv PATH to also write gdp-cross-corr-<mode>-<scan>.csv. The CSV columns match the NPZ arrays except for header_json.

Related Options

OptionUse
--output-cross-corr-npz PATHWrite the default NPZ product to a specific path.
--output-cross-corr-csv PATHAlso write the optional CSV product to a specific path.

--corr-nbin INTSet the lag-bin count.
