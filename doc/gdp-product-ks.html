GDP KS NPZ Format

KS diagnostic product saved by gdp-stats --ks.

Documentation Tree

  • Main documentation
  • gdp-setup: initialize and remember runtime and source-data locations.
  • gdp-util: read metadata from CASA gain, bandpass, or MeasurementSet tables.
  • gdp-stats: compute antenna-wise gain and bandpass products.
  • gdp-plot: plot GDP NPZ products and create missing products with gdp-stats.
  • gdp-plan-run: run GDP command intents from a plan file.

Product Formats

  • gains NPZ
  • stats NPZ
  • KS NPZ
  • self-corr NPZ
  • cross-corr NPZ

Standard Name

gdp-ks-<mode>-<scan>.npz

Example: gdp-ks-gain-scan1.npz. With --combine-scans: gdp-ks-gain-allscans.npz.

Arrays

ArrayDescription
header_jsonJSON metadata with format, version, input table, mode, data column, flag policy, channel selection, and KS convention.
scan_numbersScan axis values.
antennasAntenna axis values.
stokesStokes/polarization indices.
componentsComponent axis: real, imag.
metricsMetric axis: d_statistic, n_sample, mean, std.
ksKS array with shape (scan, antenna, stokes, component, metric). The d_statistic metric is saved in percent.
sample_countFinite sample count used for each antenna/stokes/component KS calculation.

The KS product follows the AntStat plot convention: gain mode uses finite raw real and imaginary samples; bandpass mode uses finite channel means over time; each component is tested against a fitted normal distribution.

Read This Product In Python

from pathlib import Path
import json
import numpy as np

path = Path("rundir/data-product/ks/npz/gdp-ks-gain-scan2.npz")
with np.load(path, allow_pickle=True) as z:
    header = json.loads(str(z["header_json"]))
    scans = z["scan_numbers"].astype(int)
    antennas = z["antennas"].astype(int)
    stokes = z["stokes"].astype(int)
    components = z["components"].astype(str)
    metrics = z["metrics"].astype(str)
    ks = z["ks"].astype(float)
    sample_count = z["sample_count"].astype(int)

d_i = list(metrics).index("d_statistic")
real_i = list(components).index("real")
d_real = ks[:, :, :, real_i, d_i]
print("KS D percent shape:", d_real.shape)
print("finite samples shape:", sample_count.shape)

Reading this product gives fitted-normal KS metrics with axes (scan, antenna, stokes, component, metric). The d_statistic metric is in percent, while n_sample, mean, and std describe the finite samples used in the test.
