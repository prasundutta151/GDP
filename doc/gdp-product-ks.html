GDP KS NPZ Format

KS diagnostic product saved by gdp-stats --ks.

Documentation Tree

  • Main documentation
  • gdp-setup: initialize and remember runtime and source-data locations.
  • gdp-util: read metadata from CASA gain, bandpass, or MeasurementSet tables.
  • gdp-stats: compute antenna-wise gain and bandpass products.
  • gdp-plot: plot GDP NPZ products and create missing products with gdp-stats.
  • gdp-plan-run: run GDP command intents from a plan file.

Product Formats

  • gains NPZ
  • stats NPZ
  • KS NPZ
  • self-corr NPZ
  • cross-corr NPZ

Standard Name

gdp-ks-<mode>-<scan>.npz

Example: gdp-ks-gain-scan1.npz. With --combine-scans: gdp-ks-gain-allscans.npz.

Arrays

ArrayDescription
header_jsonJSON metadata. The current implementation records the product schema.
scan_numbersScan axis values.
antennasAntenna axis values.
stokesStokes/polarization indices.
componentsComponent axis: real, imag.
metricsMetric axis: d_statistic, n_sample, mean, std.
ksKS array with shape (scan, antenna, stokes, component, metric).
